# MODULE_2_END-ASSIGNMENT

### EMPLOYEE ANALYSIS OF ABC COMPANY
------------------------------------
This is a project done for the Employee analysis of ABC company to get a comprehensive information about the employees across various teams.


-Project Overview-
------------------
This Analysis comprises:
- Preprocessing Steps
- Analysis Tasks
- Graphical Representations
- Insights

1. Preprocessing:
   - The "Height" variable was corrected by replacing the faulty entries with
     randomly generated values ranging from 150 to 180 cm.
3. Analysis Tasks:
   - Employee distribution of each teams with percentage split
     (Bar chart & Pie chart)
   - Employee segregation by position (Bar chart)
   - Predominant Age group (Bar chart)
   - Salary expenditure by team & position (Bar chart)
   - Age vs salary correlation (Scatter plot & Correlation coefficient)
4. Graphical Representations of the same given above
5. Insights




-Files-
-------
-"ABC_Employee_analysis.ipynb" : Jupyter Notebook with code, plots, and  
  explanations
-"ABC_employee_dataset.csv" : Original dataset with error
-"ABC_employee_dataset_modified.csv" : Preprocessed dataset


_How to run-
------------
-Upload the csv  & ipynb file to the colab
-Open notebook and run all the cells
